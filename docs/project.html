Final Project: Neopixel Mood-screen

(a.k.a Rina-board)

Project idea

For my final project I decided attempt to create a cosplay prop from the "Love Live Nijigasaki" series.

Design Brief

  • Project Title: Neopixel mood-screen ("Pixel-board" for short

Functions:

  • A screen made out of NeoPixel LED's that can display emotions using pixel art.
  • To be controlled via Bluetooth
  • Can be displayed and used as a cosplay prop (wearable item).

Research

Luckily there are many others who created their own pixel-board. Out of the many attempts one that stood out the most is the one done by a Japanese youtuber.

As the video documents the entire process of making his own board (and it comes with a full write up on the process), I decided I can reference his work in order to create mine.

Hence here are some specifications I came up with:

Structure

  • Stand used to display the pixel-board to be made using the CNC cutter
  • Head gear to be 3D-printed and resin casted.

Electronics

  • Using the ATTINY85, an NeoPixel strip will be arranged into a large screen which will be able to display pixel art. The display will be controlled via a Bluetooth module and the entire circuit will be powered by a battery pack and voltage regulator.
  • The NeoPixel LED strip can display 6 different pixel arts using 153 LED Lights.

Materials

Component Quantity
ATTINY85 echo board 1
ISP programmer 1
NeoPixel 2020 ultra-slim tape LED 153 LED long
Bluetooth module 1
Battery pack 1
Voltage regulator 1

Design

The entire project is split into 3 parts: Neopixel screen, Head gear, Head stand.

Neopixel Screen

This is the most important part of the project as it basically contains all the electronics. It basically is a screen made out of Neopixel LED lights.

The screen needs to be able to display certain pixel art images, so using google sheets I mapped out the LED's and made some images I wanted to be displayed and in total I will require 153 LED lights.

 

I will also need the screen to be able to store the electronic components.
